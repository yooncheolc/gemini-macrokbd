상세 설정 가이드

1. 연결 및 기본 세팅

1. 프로그램 설치 후 MKESPN K815 앱을 실행합니다.

2. 키보드가 연결되어 있어야 앱이 실행됩니다.

2. 키 설정 바꾸기

1. 프로그램을 실행하면 "Normal"과 "Game" 모드가 보입니다. 출하 상태의 키보드는 Normal = 윈도우 모드, Game = Mac 모드로 설정되어 있어요. 키보드 사용 시 빨간색 볼륨 노브를 누르면 모드가 변경 됩니다.

2. 기능 변경을 원하는 키를 클릭 후 원하는 키 기능으로 변경합니다.

3. Ctrl + C 등 키보드 키 조합 (매크로) 를 설정하고 싶으면 아래 섹션을 참고해주세요.

3. 매크로 설정하기

1. 매크로 설정을 위해 화면 상단 Macro 버튼을 눌러주세요.

2. 저희가 제공하는 매크로 사용을 위해서는 위의 다운로드 섹션에 있는 windows.dat, mac.dat, chrome.dat 파일을 받아주세요.

3. Macro 창에서 우클릭을 하면 메뉴가 나타납니다. 예를 들어 chrome.dat 매크로를 사용하고 싶으면 Import a macro group 버튼을 누른 후 chrome.dat 파일을 불러와주세요.

4. 나만의 매크로를 만들고 싶을 땐? Add a new macro 버튼을 누른 후, 매크로 폴더를 생성합니다. 그리고 Add a new macro를 눌러 새로운 매크로를 생성한 후, 하단의 플레이 버튼을 누른 후 원하는 키 조합을 직접 입력 (예: win + R, https://poin2.co.kr) 후 정지 버튼을 눌러주세요.

5. Key in macro 섹션에서 각 키 누름, 뗌의 시간을 설정해줍니다. 10~20ms 정도가 적당해요.

6. 할당을 위해 Button 메뉴로 돌아와 적용을 원하는 키를 선택하고, Macro에서 매크로 그룹과 이름을 설정합니다. 초록 체크를 눌러 저장하고, 기기에 쓰기 위해 아래의 체크 버튼을 누르고 5초만 기다려주세요.

4. 1번 키 추천 설정

1. Gemini 매크로 키패드의 1번 키는 자동으로 Gemini 앱을 열도록 프로그래밍 되어있는데요, 기기의 키보드 언어 (한/영)에 따라 종종 오작동 할 수 있습니다.

2. Windows: Win+R (실행)을 누른 후 https://gemini.google.com 으로 가게 되어있어요.
Mac: cmd + space (스포트라이트)를 누른 후 https://gemini.google.com 으로 가게 되어 있어요.

3. 대부분의 경우 동작하지만, 키보드가 한국어로 되어있을 때 종종 ㅗㅅ센://ㅎ드ㅑㅜㅑ.해ㅐ힏.채ㅡ 으로 입력되며 이동이 되지 않을 때가 있습니다.

4. 이를 방지하기 위해 크롬 브라우저에서 한가지 설정을 해주시면 좋아요. 바로 gem (또는 ㅎ드)을 치면 바로 Gemini로 접속하게 해주는 설정이죠.

5. 크롬 브라우저의 검색엔진 설정 (chrome://settings/searchEngines) 에 들어가서 사이트 검색 부분에 아래 항목을 추가해줍니다.
[이름: Gemini / 단축키: gem / URL: https://gemini.google.com]
[이름: Gemini / 단축키: ㅎ드 / URL: https://gemini.google.com]

6. 이제 브라우저가 켜진 상태에서, 1번 키를 누르면 새탭 (ctrl + t), gem, enter 이렇게 치는 매크로를 만들어주면, 크롬 브라우저가 켜져있을 때 언제든 1번 키를 누르면 빠르게 Gemini App을 켤 수 있게 됩니다.
